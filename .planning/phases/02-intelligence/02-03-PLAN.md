<task type="auto">
  <name>Task 1: Implement lifecycle automation module</name>
  <files>src/core/lifecycle.ts</files>
  <action>
Create `src/core/lifecycle.ts`:

1. **`checkKrTaskCompletion(sevenDir: string, krId: string): Promise<{allDone: boolean, total: number, done: number}>`**
   - List all tasks, filter by parentId === krId
   - Return counts and whether all are "done"

2. **`runMeasureScript(script: string): Promise<{stdout: string, exitCode: number}>`**
   - Validate script: MUST start with `npm run` or be a path under `.7even/scripts/` or `./scripts/`. Throw error otherwise (security per RESEARCH pitfall 3).
   - Use `execSync(script, { encoding: "utf-8", timeout: 30000, cwd: getGitRoot() })`
   - Return stdout and exitCode (0 = success)
   - Catch errors: return err.stdout + err.status

3. **`evaluateKr(sevenDir: string, krId: string): Promise<{status: "achieved"|"needs-breakdown", scriptOutput?: string}>`**
   - Read KR data
   - If KR has measureScript: run it via runMeasureScript
     - exitCode 0 → "achieved"
     - exitCode non-0 → "needs-breakdown"
   - If no measureScript: return "needs-breakdown" with note that agent should interpret manually
   - Does NOT auto-transition KR status — returns recommendation. Per CONTEXT.md: agent prompts "ready to evaluate?" first, never silently transitions.

4. **`cascadeAchievement(sevenDir: string, krId: string): Promise<{krMoved: boolean, objectiveMoved: boolean, objectiveId?: string}>`**
   - Move KR to "achieved" via moveItem
   - Read KR to get parentId (objective)
   - Call checkObjectiveCompletion
   - If all KRs achieved, move objective to "achieved"
   - Return what was moved

5. **`checkObjectiveCompletion(sevenDir: string, objectiveId: string): Promise<{allAchieved: boolean, total: number, achieved: number}>`**
   - List all KRs, filter by parentId === objectiveId
   - Return counts and whether all are "achieved"

Important: Use a guard flag pattern to prevent circular lifecycle triggers (RESEARCH pitfall 2). Export `let _lifecycleGuard = false` and check/set it in cascadeAchievement. Reset after completion.
