2. **`.opencode/commands/7-breakdown.md`:**
   - Description: "Break a key result into tasks with MECE validation"
   - Shell injections: `npx 7 key-result show $1`, `npx 7 task list`, `npx 7 session list --status active`
   - Agent instructions: same session flow but for KR→task decomposition, cross-reference existing tasks for overlap (OKR-07)

3. **`.opencode/commands/7-evaluate.md`:**
   - Description: "Evaluate a key result's completion and cascade achievements"
   - Shell injections: `npx 7 evaluate kr $1`, `npx 7 key-result show $1`
   - Agent instructions: review task completion, run evaluation, interpret results, prompt for stakeholder approval, cascade if met, trigger new breakdown session if not met

4. **`.opencode/commands/7-start-task.md`:**
   - Description: "Start a task with acceptance criteria definition (TDD workflow)"
   - Shell injections: `npx 7 task show $1`, `npx 7 key-result show <parentId>`, `npx 7 objective show <grandparentId>`
   - Agent instructions: Guide human through defining acceptance criteria BEFORE implementation. Each criterion needs executable script reference. Write criteria to task JSON via `npx 7 task update`. Move task to in-progress only after criteria approved. Per CONTEXT.md: "agent leads — humans are bad at TDD"

5. **`.opencode/commands/7-pause.md`:**
   - Description: "Pause current task work"
   - Shell injections: `npx 7 task list --status in-progress`
   - Agent instructions: record current state in task comments, note what's left, keep task in-progress

6. **`.opencode/commands/7-proceed.md`:**
   - Description: "Resume paused task work"
   - Shell injections: `npx 7 task show $1`, `npx 7 estimate suggest $1`
   - Agent instructions: load task context, show acceptance criteria progress, suggest re-estimation, continue implementation

7. **`.opencode/commands/7-finish.md`:**
   - Description: "Finish a task — run acceptance scripts and complete"
   - Shell injections: `npx 7 task show $1`, `npx 7 evaluate kr <parentId>`
