Custom div-based Gantt (Recharts doesn't support Gantt — per research).

**GanttHeader.tsx:**
- Horizontal time axis with day/week tick marks
- Computes visible date range from data min/max dates with padding
- "Today" marker as vertical line
- Zoom controls: day/week/month granularity

**GanttBar.tsx (per research pattern):**
- Position: absolute left/width as percentage of total timeline range
- Color: objective's assigned color (hash UUID to palette index per research pitfall 4)
- KR bars: derived shade of parent objective color (oklch lightness offset)
- Task bars: lighter shade, thinner height
- Progress fill: darker overlay proportional to `progress` field
- Hover tooltip: name, dates, SP remaining
- Height hierarchy: objectives 32px, KRs 24px, tasks 18px
- Sharp edges (no border-radius)

**GanttChart.tsx:**
- Fetches `/api/metrics/gantt` for bar data
- Left column: item names with indentation (objective → KR → task)
- Right area: scrollable timeline with GanttBars
- Objectives are expandable rows — click to show/hide child KRs and tasks
- Default: objectives expanded, tasks collapsed under KRs
- 3px LCARS decorative bars between objective groups
